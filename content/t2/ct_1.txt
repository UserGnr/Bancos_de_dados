SQL

- SQL (Structured Query Language ou Linguagem de Consulta Estruturada)
- Inspirada pela Álgebra Relacional
- Linguagem padrão dos bancos de dados relacionais, contudo, dependendo do banco de dados pode haver uma mudança no dialeto
- 5 subgrupos:
    - DQL (Data Query Language ou Linguagem de Consulta de Dados)
    - DML (Data Manipulation Language ou Linguagem de Manipulação de Dados)
    - DDL (Data Definition Language ou Linguagem de Definição de Dados)
    - DCL (Data Control Language ou Linguagem de Controle de Dados)
    - DTL (Data Transaction Language ou Linguagem de Transação de Dados)

- Por padrão, os comandos SQL são todos em letras maiúsculas e o que for referente aos dados/tabelas estarão em minúsculo


DQL - Data Query Language
- Consultas no banco de dados
- É o mais utilizado
- Possui apenas 1 comando, porém existem várias cláusulas e opções
- É possível fazer buscas complexas
- Comando: SELECT
Exemplo:
    SELECT * FROM tipos_produto;
        Comentário:
        - Todos os dados da tabela 'tipos_produto' será selecionado
        - O asterisco '*' indica que se quer todos os campos da tabela
        - O comando termina com o ponto e vígula ';' no final
    
    SELECT codigo, descricao FROM tipo_produto;
        Comentário:
        - Seleciona todos os dados da tabela 'tipos_produto'
        - Os únicos campos especificados são 'codigo' e 'descricao'
    
    SELECT codigo AS cod, descricao AS desc FROM tipo_produto AS tp;
        Comentário:
        - Apelidando (alias) os campos e a tabela

DML - Data ManipulationLanguage
- Utilizado para realizar inclusões, alterações e exclusões de dados em registros do banco de dados
Comandos:
    INSERT (inserir um registro em uma tabela existente)
    UPDATE (alterar valores de dados em um ou mais registros de uma tabela)
    DELETE (remover registros de uma tabela)
Exemplo:
    INSERT INTO tipos_produto (descricao) VALUES ('Notebook');
        Comentário:
        - INSERT INTO: insere na tabela- 
        - 'tipos_produto' é a tabela
        - 'descricao' é a coluna
        - 'Notebook' é o valor
    
    INSERT INTO produtos (descricao, preco, codigo_tipo_produto) VALUES ('Notebook', 1200, 1);
        Comentário:
        - Insere na tabela 'produtos' na coluna 'descricao' o valor 'Notebook'
        - Insere na tabela 'produtos' na coluna 'preco' o valor 1200
        - Insere na tabela 'produtos' na coluna 'codigo_tipo_produto' o valor 1
